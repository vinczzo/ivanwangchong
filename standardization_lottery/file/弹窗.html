弹窗按钮汇总：（按钮文案由运营提供，此处仅描述功能）

1. “去逛逛”：可有运营提供链接，点击后跳转至指定链接

2. “继续抽奖”：点击后关闭弹窗并直接出发抽奖动作；该按钮仅可在有抽奖机会或积分足够抽奖时展示

3. “领取奖品”：点击后校验用户填写的收货信息是否完整，若完整则提交收货信息，否则toast提示“请填写正确的收货地址信息”；该按钮仅可在实物奖中奖弹窗中出现

4. “关闭”：点击后关闭弹窗
